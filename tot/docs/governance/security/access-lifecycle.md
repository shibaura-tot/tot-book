# 権限の付与・剥奪（入退会フロー）

## 入部時（担当：運用）
- [ ] Bitwarden招待（個人アカウント＋2FA確認）
- [ ] 必要最小のボルトへ**閲覧**付与
- [ ] 公開Wiki/連絡網への追加

## 役職就任時
- [ ] 対象ボルトの権限を**編集/管理**へ昇格  
- [ ] 重要アカウントの2FA再確認

## 退会/役職交代時（即日）
- [ ] 全ボルトから除外、共有から解除
- [ ] 重要アカウント**ローテーション**
- [ ] 連絡網/公開プロフィール更新
